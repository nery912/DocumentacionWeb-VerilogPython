Introducción a MyHDL

MyHDL es una biblioteca de Python de código abierto que redefine la descripción de hardware. Te permite modelar y verificar lógica digital compleja utilizando la sintaxis limpia de Python, para luego compilar automáticamente tu diseño a Verilog o VHDL optimizado y listo para síntesis en FPGAs.

📦 1. Configuración del Entorno e Instalación ↓

Para empezar, necesitas tener instalado Python (versión 3.x o superior). Podés gestionar la instalación de la librería usando el gestor tradicional o herramientas modernas de alto rendimiento:

# Opción estándar con PIP: pip install myhdl
# Opción rápida con UV (Recomendado): uv pip install myhdl
💡 2. Implementación: Un Intermitente LED (Blinker) ↓

El circuito para hacer parpadear un LED es el "Hola Mundo" de las FPGAs. Usaremos una señal de reloj (clock) para sincronizar eventos e incrementar un contador que conmute el estado del LED a intervalos específicos:

from myhdl import block, always_seq, Signal, intbv

@block
def led_blinker(clock, led):
    # Contador de 24 bits para controlar la velocidad del parpadeo
    counter = Signal(intbv(0)[24:])  
    
    @always_seq(clock.posedge, reset=None)
    def blink():
        # Conmuta el LED al alcanzar el valor máximo del contador
        if counter == counter.max - 1:
            led.next = not led
            counter.next = 0
        else:
            counter.next = counter + 1
            
    return blink
🧪 3. Ejecutar una Simulación Nativa ↓

MyHDL te permite simular y verificar tu diseño directamente desde scripts de Python, eliminando por completo la necesidad de abrir pesados entornos de software de FPGA en las primeras fases de prueba:

from myhdl import Simulation, delay, instance

# Definición de señales lógicas
clock = Signal(0)
led = Signal(0)

# Instanciamos nuestro módulo blinker
blink_inst = led_blinker(clock, led)

# Generador de estímulo de reloj asincrónico
@instance
def clock_gen():
    while True:
        clock.next = not clock
        yield delay(10)  # Invierte el reloj cada 10 unidades de tiempo

# Lanzamos el motor de simulación interno
sim = Simulation(blink_inst, clock_gen)
sim.run(200)  # Corre por 200 unidades de tiempo
